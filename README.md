# WanderLens - Interactive Travel Image Gallery

## 📌 Project Overview

WanderLens is a modern and responsive travel image gallery developed as part of the **CodeAlpha Frontend Development Internship - Task 1**.

The project allows users to explore beautiful travel destinations through an interactive gallery featuring image previews, fullscreen lightbox viewing, navigation controls, and a professional user interface.

---

## 🚀 Features

* Responsive Design for Desktop, Tablet, and Mobile
* Modern Travel-Themed User Interface
* Interactive Image Gallery
* Fullscreen Lightbox View
* Previous and Next Image Navigation
* Smooth Hover Effects and Animations
* About Popup Modal
* Contact Popup Modal
* Fixed Navigation Bar
* Professional Footer Section

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JS)
* Google Fonts (Poppins)
